# MakeMyTrip Automation Framework

Selenium automation framework for MakeMyTrip (Flights).

## Tech Stack
- Python
- Selenium
- Page Object Model (POM)

## Features Automated
- From / To city selection
- Date selection
- Traveller & class selection
- Flight search

## How to Run
```bash
pip install -r requirements.txt
python tests/test_flight_search.py
